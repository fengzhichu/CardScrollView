# CardScrollView
用 swift 写的一个卡片横向滚动的控件。
![Demo](http://screenshot.net/gkln3a5.jpg)

## 特性

- 可以自定义每个卡片的高度和宽度；
- 可以自定义卡片之间的间距；
- 根据卡片的宽度分配卡片的个数；
- 支持卡片的复用，提高性能；
- 支持动画效果。
